# 🔐 Password Generator App

A modern and customizable password generator built with **React** and **Tailwind CSS**.
Generate strong, secure passwords with options for uppercase, lowercase, numbers, and special characters.

---

## 🚀 Features

* 🔢 Include numbers (0–9)
* 🔤 Include uppercase letters (A–Z)
* 🔡 Include lowercase letters (a–z)
* 🔣 Include special characters (!@#$%^&*)
* 🎚️ Adjustable password length
* 📋 One-click copy to clipboard
* ⚡ Instant password generation
* 🎨 Clean UI with Tailwind CSS

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Styling:** Tailwind CSS
* **State Management:** React Hooks (`useState`)

## 🧩 How It Works

1. Select password length
2. Choose character types:

   * Uppercase letters
   * Lowercase letters
   * Numbers
   * Special characters
3. Click **Generate Password**
4. Copy the password with one click
